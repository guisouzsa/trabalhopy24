# Trabalho de Informática Básica e Planejamento de Carreira

## Sobre o projeto

Este repositório reúne atividades desenvolvidas como parte de um trabalho escolar de informática básica e planejamento de carreira. O foco principal foi a introdução à programação utilizando Python, com exercícios voltados para lógica de programação.

O projeto contém diferentes questões práticas, cada uma abordando conceitos fundamentais como entrada de dados, condições, loops e saída de informações.

## Objetivo

O objetivo do trabalho é desenvolver habilidades iniciais em programação, lógica computacional e resolução de problemas, além de introduzir o aluno ao uso de Python como linguagem de aprendizado.
