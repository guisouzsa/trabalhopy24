# Trabalho de Informática Básica e Planejamento de Carreira

## Sobre o trabalho

Este repositório reúne atividades desenvolvidas como parte de um trabalho escolar de informática básica e planejamento de carreira. O foco principal foi o aprendizado de programação utilizando Python, com exercícios voltados para lógica de programação.

O trabalho contém diferentes questões práticas, cada uma abordando conceitos fundamentais como entrada de dados, condições, loops e saída de informações.

## Objetivo

O objetivo deste trabalho é desenvolver habilidades iniciais em programação, lógica computacional e resolução de problemas, além de introduzir o uso da linguagem Python.
